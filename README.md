Overview
This project uses a small subset of the Netflix Prize dataset to explore rating prediction and basic recommendation modeling. Each record represents a user rating for a movie.

Dataset

ratings-train.txt and ratings-test.txt: rows formatted as MovieID,UserID,Rating

Ratings are from 1 to 5

titles.txt: rows formatted as MovieID,Title 

description

Files in this repo

Netflix_rating_solution.ipynb: main notebook (data loading, modeling, evaluation)

ratings-train.txt: training set

ratings-test.txt: test set

titles.txt: movie titles

How to run

Download the repo

Open Netflix_rating_solution.ipynb in Jupyter Notebook / JupyterLab

Run cells from top to bottom

Notes
This dataset is a subset of the Netflix Prize data.
